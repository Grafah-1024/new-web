Anchor Tag

An anchor (<a>) tag is used to create hyperlinks that connect a web page to another section on the same webpage, to another webpage, or to allow one to upload or download a file.

The <a> tag has an href attribute that specifies the destination or where the user will be taken upon clicking it. A hyperlink can be text, an image, a video, etc.

General syntax of an anchor tag
<a href="destination">target destination</a>

HTML Lists

A list is a sequence of items arranged in a particular order, which can be horizontal or vertical. HTML lists organize items into ordered lists, unordered lists, or definition lists to make information clear and easy to read.

Types of lists in HTML: 1. Ordered list 2. Unordered list 3. Definition list.

Ordered List

In an ordered list, sequence of items are labeled using numbers, letters, or roman numerals.

  1. Item 1
  2. Item 2

Form Tag

HTML forms are used to get input from the user to the web application. These inputs are processed by the business logic and may be stored in the web database.

Task 1

Create a list of all components of HTML forms (using a definition list to capture the part name, its purpose, and its attributes).

Solution 1
<form>
The container for all form elements. Key attributes: action, method (GET/POST).
<input>
The most common form element used to capture user data. Key attributes: type, name, value, placeholder, required.
<label>
Defines a label for form elements to improve accessibility. Key attribute: for (matches the input's id).
<textarea>
Used for multi-line text input. Key attributes: rows, cols.
<button>
A clickable button to submit, reset, or perform an action. Key attribute: type (submit/reset/button).
